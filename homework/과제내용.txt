PC버전일떄는 2단으로 구성하되 모바일때는 1단으로 구성

PC버전이나 모바일버전이나 리사이즈될때마다 이미지 사이즈랑 폰트등 크기변경

1차 작업은 검색차트  예시작업 https://m.naver.com/

기능이나 구현은 m.naver.com 을 보면서 구현하면됨

1. 검색바는 항상 탑으로 올라가게 한다. (스크롤내리면 메뉴는 올라가있게)

2. 검색바를 누르면 검색레이어 팝업호출 

※ 레이어팝업을 호출하면 뒷편에 내용들이 보이지 않게처리

힌트 : height 를 꽉채워야함

3. 컨텐츠들은 한개의 섹션대로 작업하기 

ex) 검색어트렌드 , 데이터랩바로가기 , 급상승검색어 , 실시간검색어 등등

[검색어트렌드]
4. 검색어트렌드에서 검색어를 치다가 X누르면 누른 검색어 삭제

5. 느낌표를 클릭하면 레이어팝업 호출

6. 검색어 트렌드 및에 탭은 양쪽으로 구성하되 비활성화 탭을 눌러서 on이되면 

왼쪽으로 활성화시켜서 분류한다. 

닫기를 누르면 다시 비활성화 우측으로 이동후 활성화 끄기

7. 검색어 트렌드 탭을 클릭시 차트 api 를 이용해서 임의적인 데이터차트를 구현

1개월 , 3개월 , 1년을 눌럿을때 데이터만 변경하여 차트를 갱신

8.  초기화를 누르면 X 모든 초기 설정으로 돌리기

힌트 :  클릭하면 css로 on 효과를 미리 만들어놓고 눌럿을떄 클래스를 추가해서 on효과추가

[실시간검색어]

1. 1초간격으로 실시간검색어 순위를 변경시킨다.

2. 해당탭을 눌럿을때 모든 기능은 멈추고 5초뒤 다시 실시간검색어 기능 시작

3. 실시간 검색어 탭이 활성화 되었을때 on 효과를 추가

4. 일간검색어 , 실시간 검색어 급상승토픽등 더보기를누르면 숨겨진 컨텐츠 보여주기


[급상승검색어]

1. 해당 5구간의 클릭점을 잡아놓고 눌럿을때 액션주기


[공통]

1. 폰트키우기 버튼을 누르면 모든 폰트 키우기 

2. 폰트키우기 버튼을 누르면 모든 폰트 줄이기

3. 모든폰트 키운상태에서 리사이즈하면 원래 적용한상태의 폰트로 다시 돌아오기
